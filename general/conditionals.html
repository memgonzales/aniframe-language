Conditionals


The syntax for conditionals is as follows:

if <condition>:
    <expression>
else if <condition>:
    <expression>
else:
    <expression>

The else if and else clauses are optional. There can be multiple else if clauses, but there can only be at most one else clause.

Example:

num: Number = 401

ctr: Number = 25
                
if num > 400 || num < 0:
    info("trigger me!")
    info(num)

else:
    info(ctr)

Output:
trigger me!
401
