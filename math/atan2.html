atan2()


Returns the arctangent of a value
The value returned is between -π and π

atan2(y, x)
y y-coordinate
x x-coordinate

Example:

x: Number = 1
y: Number = -1
marker_y = 30 * atan2(y, x) + 300
                                                                                                                
marker = circle(x, marker_y, 200)
marker.draw(1, 1000)
